# **Mac Evidence Processing Cheat Sheet (Improved)**

## **1. Imaging & Documentation**

- [ ]  Acquire physical image of HDD/SSD.  
    Name format: **{caseNumber}_Ex{exhibit}.e01**  
    Save to: **\case{caseNumber}\Ex{exhibit}\Image\**
- [ ]  Preserve acquisition logs.  
    Add imaging log to ForensicCases.xlsx:  
    `python ForensicsReporter.py -l`
- [ ]  Photograph/record system state (logged‑in user, running processes, attached peripherals, encryption state).
- [ ]  Document FileVault status:  
    `fdesetup status`  
    (Record whether decrypted image was obtained.)

---

## **2. Triage of User Activity**

- [ ]  Collect and review **~/.bash_history** (or **zsh_history** for newer macOS).
- [ ]  Extract: - **LaunchAgents** and **LaunchDaemons**  
    /Library/LaunchAgents/, /Library/LaunchDaemons/, ~/Library/LaunchAgents/ - **Login items**  
    System configuration dump or plist review
- [ ]  Check for persistence mechanisms (cron, plist, profiles).

---

## **3. Accounts & Authentication**

- [ ]  Review **/etc/passwd** and **/etc/shadow** (or macOS equivalent: `/var/db/dslocal/nodes/Default/users/*.plist`)
- [ ]  Parse user .plist files for account metadata, timestamps, aliases.
- [ ]  Extract password hashes (if available).
- [ ]  Crack hashes (optional) using **John the Ripper** or **Passware**.

---

## **4. Browser & Credential Artifacts**

- [ ]  Extract browser-stored passwords (Keychain-dependent).  
    Export to **passwords.xlsx**.
- [ ]  Extract and process: - Safari database files (History.db, Downloads.plist) - Chrome/Brave/Edge SQLite histories - Firefox profiles (places.sqlite, logins.json)
- [ ]  Export web history logs as:  
    **WebHistory_{caseNumber}_{caseName}_{exhibit}.xlsx**

---

## **5. System Logs & Metadata**

- [ ]  Parse unified logs (/var/db/diagnostics/, log archives).
- [ ]  Review Spotlight metadata:  
    `/Volumes/<image>/_.Spotlight-V100/`
- [ ]  Examine TCC database (privacy permissions).
- [ ]  Parse QuarantineEventsV2 for downloaded files.
- [ ]  Review recent items lists (applications, servers, docs).

---

## **6. User Data & Communications**

- [ ]  Extract Messages (iMessage databases), Facetime logs.
- [ ]  Review Mail directories.
- [ ]  Check for cloud sync artifacts: - iCloud - Dropbox - OneDrive - Google Drive

---

## **7. Additional Artifacts**

- [ ]  Application usage logs (knowledgeC.db)
- [ ]  Locations & movement (locationd logs, caches)
- [ ]  AirDrop logs
- [ ]  USB device history (IOUSBHost, system profiler dumps)
- [ ]  Time Machine backups (local & remote)

## **8. mac_apt.exe (FAST Plugin Export)** s:\tools\mac_apt\

-  Prepare your case directory structure:  
    **\case\{caseNumber}\Ex{exhibit}\mac_apt\**
    
-  Run mac_apt.exe against the mounted image or extracted APFS container.  
    Example command (write this into your cheat sheet as you prefer):
    
    ````
    ```
    mac_apt.exe -i <path_to_image_or_volume> -o <output_folder> FAST
    ```
    
    Notes:  
    - `-i` can point to the E01 mount point, raw disk, or the APFS container.  
    - `FAST` runs the streamlined artifact set (QuickLook, KnowledgeC, RecentItems, USB, TCC, Spotlight, etc.).  
    ````
    
-  Ensure mac_apt output folders are created:  
    **Artifacts**, **Databases**, **HTML_Reports**, **Logs**.
    
-  Convert mac_apt SQLite artifact data to Excel (OPTIONAL):  
    `mac_apt.exe -x <mac_apt_output_folder>`
    
-  Add FAST output to your case materials:  
    Save to: **\case\{caseNumber}\Ex{exhibit}\mac_apt\FAST\**
